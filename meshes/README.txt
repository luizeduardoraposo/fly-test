Coloque aqui seus arquivos .glb para o jogo Babylon.js.

Sugestão de nomes para os modelos:
- spaceship.glb  (nave espacial)
- mountain.glb   (montanha)
- building.glb   (prédio)
- tree.glb       (árvore)

Sugestões de fontes gratuitas/open source:
- Nave: https://quaternius.com/packs/spaceships.html
- Montanha: https://kenney.nl/assets/nature-kit
- Prédio: https://kenney.nl/assets/city-kit
- Árvore: https://kenney.nl/assets/nature-kit

Basta baixar, converter para .glb se necessário, e colocar aqui com o nome correto.
Se o arquivo não existir, o jogo usará uma primitiva como fallback.
